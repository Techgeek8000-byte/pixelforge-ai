# PixelForge AI — Free AI Image Generator

Generate stunning AI images, avatars, logos, wallpapers, and more — completely free.

🌐 **[pixelforge-ai-chi.vercel.app](https://pixelforge-ai-chi.vercel.app)**

## Features

- **AI Image Generation** — Text-to-image, AI Avatars, Logo Generator, Wallpaper Creator, Thumbnail Maker, AI Art Studio, Social Media Graphics, Business Card Designer
- **Multiple styles** — Photorealistic, Anime, Digital Art, Oil Painting, 3D Render, Pixel Art, and more
- **Multiple sizes** — Square (1:1), Landscape (16:9), Portrait (9:16), and custom dimensions
- **Prompt & Result view** — Toggle between your prompt and the generated image
- **10 free generations/day**
- **Pro plans** — $1/week, $2/month, $12/year, or $25 lifetime (unlimited generations, no watermark)
- **Dark theme** — Modern, immersive UI

## Tech Stack

- [Next.js](https://nextjs.org/) (App Router)
- TypeScript
- Tailwind CSS
- Zustand
- Framer Motion
- Lucide React
- Sonner
- LemonSqueezy (payments)
